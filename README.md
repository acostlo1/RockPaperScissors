🎮 Rock Paper Scissors (Java Console Game)

A simple console-based Rock Paper Scissors game built in Java. The user plays against the computer, which generates random moves. The game includes input validation and a replay feature for continuous gameplay.

🚀 Features
Play Rock, Paper, Scissors against the computer
Randomized computer moves using Random
Input validation for user choices
Replay option to play multiple rounds
🧠 Concepts Used
Arrays
Loops (do-while)
Conditional statements (if/else)
User input with Scanner
Random number generation (Random)
▶️ How It Works
User enters a move (rock, paper, or scissors)
Computer randomly selects a move
The program determines the winner
User can choose to play again or exit
🛠️ How to Run
Compile the program:
javac Main.java
Run the program:
java Main
📈 Future Improvements
Add score tracking (wins/losses/ties)
Improve input handling (shortcuts like r/p/s)

💬 Notes

This project was created to practice Java fundamentals including loops, conditionals, arrays, and user input handling.
